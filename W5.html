W4 Findings Made as User Journey Map

Persona: Efficiency-Oriented Shopper

1. Pre-Visit

Actions:
  • Makes a list
  • Checks prices online
Emotion:

Focused / Neutral

Pain Point:

Uncertainty about stock availability

2. Entering Store

Actions:
  • Scans layout
  • Looks for aisle signs
Emotion:

Mild Confusion

Pain Point:

Store layout changes

3. Searching Items

Actions:
  • Walks multiple aisles
  • Asks for help
Emotion:

Frustration

Pain Point:

Difficult navigation

4. Checkout

Actions:
  • Chooses self-checkout or cashier
  • Waits in line
Emotion:

Impatient

Pain Point:

Long wait times

5. Exit

Outcome:

Satisfaction (if smooth) or Stress (if frustrating)

Key Insights

  • Frustration increases when navigation is unclear.
  • Efficiency matters more than browsing.
  • Checkout strongly impacts overall satisfaction.
  • Predictability reduces shopper stress.
